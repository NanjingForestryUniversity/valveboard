# 发送端实现举例

## 概述

按照阀板程序，这里为基于三个平台的发送端实现

- stm32f103

  VScode工程，参考https://github.com/3703781/vstm32code

- zynq7100

  实现了发送模块，可直接例化
  
- epc9600

  基于嵌入式linux的程序，线程安全，具体板子为EPC-9600I-L，是广州致远电子有限公司开发的基于AM335x系列处理器的工控主板，处理器内核为800MHz的Arm Cortex-A8。
